## Current controls

Select a PNG image, then use **Open puppet window** to show it in a separate window with a transparent background. Adjust **Puppet size** with the slider or +/- buttons. Clear **Puppet window unlocked** to remove the title bar and make the puppet window click-through and fixed in place. Enable **Always on top** to keep the puppet above other windows.
