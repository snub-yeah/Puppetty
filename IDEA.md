# Puppetty Idea

The idea for Puppetty is to make a program that acts as a PNGTuber program. It allows for the user to select an image, then the image will bounce up and down with their microphone activity like a puppet show

It is inspired by BattleBlock Theater cutscenes, where characters bounce while the narrator talks

Puppetty will have two main windows:

    A configuration window where you can change settings and the image
    The actual puppet window which is transparent and can be clicked through whenever locked. This is to allow you to overlay it for streaming or videos
